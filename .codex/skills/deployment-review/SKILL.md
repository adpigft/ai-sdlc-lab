# Deployment Review Skill

## Purpose
Review deployment approach, rollback, migration, feature flags, environment readiness, and monitoring.

## When to use
Use when a release deployment plan is ready for review.

## Inputs
- deployment plan
- validation report
- release notes
- traceability

## Process
1. Review deployment and rollback approach.
2. Review migration, feature flags, and monitoring.
3. Review environment readiness and risks.
4. Return deployment review findings.

## Outputs
- deployment review findings

## Quality checks
- deployment readiness
- rollback readiness
- migration readiness
- feature flag readiness
- environment readiness
- monitoring readiness

## Human gate
Human approval is required before production deployment.
