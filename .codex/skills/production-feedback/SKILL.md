# Production Feedback Skill

## Purpose
Capture production feedback, user issues, support tickets, incidents, and enhancement ideas.

## When to use
Use when production signal needs to be recorded and converted into improvement work.

## Inputs
- production feedback
- support tickets
- incidents
- user issues
- enhancement ideas

## Process
1. Capture the feedback signal.
2. Classify the issue or idea.
3. Link to affected artifacts or work items.
4. Return feedback entries and improvement backlog.

## Outputs
- feedback entries
- improvement backlog

## Quality checks
- traceability to source
- actionable classification
- ownership identified

## Human gate
Human review is required before backlog prioritization.
