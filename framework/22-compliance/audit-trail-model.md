# Audit Trail Model

## Purpose

This model defines the minimum audit trail expected for enterprise AI-SDLC governance.

## Audit Questions

- Who did what?
- When did they do it?
- Using which skill?
- Using which model?
- Using which context?
- In which workflow state?

## Retention Rules

- retain audit records according to regulatory and organizational policy
- keep records immutable where required
- make the audit trail available for review and evidence collection

## Notes

- Audit trail requirements should be additive and traceable.
- The model should support internal review, external audit, and incident investigation.
