# AgentOps Scripts

This folder is reserved for future lightweight AgentOps helpers.

## Planned Scripts

- `collect-agent-run-log.py`
- `summarize-agent-feedback.py`
- `detect-agent-rework-patterns.py`
- `generate-agentops-metrics.py`

## Purpose

These scripts will eventually support:

- collecting run metadata
- summarizing human feedback
- finding repeated rework patterns
- generating operational metrics

## Scope

- documentation and scaffolding only for now
- no external systems
- no secrets
- no automatic writes to Jira, Confluence, or GitHub

## Notes

- Git remains the source of truth for any future AgentOps definitions and output contracts.
