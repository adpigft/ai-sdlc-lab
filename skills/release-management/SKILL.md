# Release Management Skill

## Purpose
Coordinate release scope, approvals, validation evidence, rollback, and production readiness.

## When to use
Use when release coordination and release controls are needed.

## Inputs
- validation evidence
- release notes
- approvals
- rollback plan
- traceability

## Process
1. Confirm scope and evidence.
2. Confirm approvals and rollback readiness.
3. Confirm production readiness.
4. Return a release management plan.

## Outputs
- release management plan

## Quality checks
- approval status
- validation evidence
- rollback readiness
- production readiness

## Human gate
Human approval is required before release execution.
