# Operational Readiness Skill

## Purpose
Validate runbooks, dashboards, alerts, support procedures, SLAs, and production support handover.

## When to use
Use when operational readiness must be confirmed before release.

## Inputs
- runbooks
- dashboards
- alerts
- support procedures
- SLAs
- release notes

## Process
1. Review support and monitoring readiness.
2. Review handover and SLAs.
3. Review operational gaps and risks.
4. Return an operational readiness report.

## Outputs
- operational readiness report

## Quality checks
- runbooks
- dashboards
- alerts
- support procedures
- SLA coverage
- production handover

## Human gate
Human approval is required before production support handover.
